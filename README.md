# Weather App

## Objective
This project is a simple Weather Application developed using Python.

## Tools Used
Python
API (for weather data)

## Steps Performed
1. Asked the user to enter a city name.
2. Connected to a weather API to get current weather data.
3. Displayed temperature, weather condition, and other details.

## Outcome
The application provides real-time weather information for any city entered by the user.
